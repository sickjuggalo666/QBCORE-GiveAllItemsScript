# GiveAllItemsScript

A lightweight QBCore script that allows administrators to give a selected item to all online players.

## Important Information
- This resource includes an `fxmanifest.lua` and a `server.lua`
- Permissions can be changed inside `server.lua`
- DO NOT rename or change any file or folder names  
  Renaming `fcg-giveallitems` or any files will cause the script to break

## Requirements
- QBCore
- ox_inventory  
  (This script will NOT work with other inventory systems)

## Installation
1. Download or clone this repository
2. Place the `fcg-giveallitems` folder into your `resources/[qb]` directory
3. Add the following to your `server.cfg`:
   ensure fcg-giveallitems
4. Restart your server

## Configuration
- Open `server.lua`
- Edit the permission check to match your server’s staff roles or ACE permissions

## Notes
- This script is designed ONLY for ox_inventory
- Changing file or folder names WILL break the resource
